# Webpage Tracker

A Python-based automation tool that captures webpage snapshots, extracts structured information, compares changes over time, and exports differences to Excel.  
Useful for monitoring People/Team pages, directory listings, or any webpage where content changes matter.

---

## 🚀 Features

- Capture HTML snapshots using Selenium (headless Chrome)
- Extract structured data (Name + Designation) using BeautifulSoup
- Compare old vs new snapshots
- Detect Added / Removed entries
- Export changes to an Excel file
- Automatically timestamps each run
- Simple wrapper function to automate the entire workflow
